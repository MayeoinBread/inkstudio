# picpak-open
Open-source app for Picpak hardware

# Objectives
- Provide alternative application for interacting with a Picpak ePaper photo frame
- Alternative dithering algorithms
- Cropping/resizing of selected images
- Reverse-engineer panel interface to determine functionality
- Preview image rendering on panel

# Progress
- Flutter app boilerplates
- Image import to Flutter app
- Processing image using Floyd-Steinberg or Atkinson dithering algorithms
- Live preview of processed image
